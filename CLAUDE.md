# Jeison Christ — Developer Portfolio

## Stack
React + TypeScript + Vite + Tailwind CSS + GSAP ScrollTrigger

## Referencia visual
landonorris.com — editorial, premium, atlético. Cada sección cambia de color de fondo al scrollear.

## Paleta
- Cream: #F2EDE4
- Olive: #1C2B1A
- Black: #0D0D0D
- Lime: #C5FF4A
- Text: #111111

## Tipografía (Google Fonts)
- Cormorant Garamond — headlines serif
- DM Sans — body y labels
- DM Mono — email

## Secciones
1. Hero — cream, figura cutout sin fondo, nombre "JEISON CHRIST" grande serif
2. Editorial — olive, foto grid asimétrico + quote
3. Projects — cream, 3 cards con tech chips
4. Contact — negro, email grande monospace

## Animaciones GSAP
- Fade + slide up staggered al cargar el hero
- Background color transiciona suavemente entre secciones con ScrollTrigger
- Cards de proyectos aparecen al entrar al viewport

## Notas
- El nombre siempre es "JEISON CHRIST"
- Mobile responsive
